
## 本项目fork自https://github.com/rentianyu/Ad-set-hosts

## 1. 简介

- 本项目集合多个去广告 `hosts`
- 本项目已实现GitHub Actions 自动更新，每4小时更新一次
- 仅供学习交流和自用，请勿商用，如有侵权，请联系删除

## 2. 使用方法与版本介绍

- 安卓直接替换`/system/etc/hosts`，或用 `Magisk` 模块刷入
- 电脑勿用，`hosts` 文件过大会造成电脑卡顿
